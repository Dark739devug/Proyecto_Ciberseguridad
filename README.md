# 📊 Sistema de Facturación

Bienvenido al repositorio del **Sistema de Facturación**.
Este proyecto está diseñado para gestionar de forma eficiente el control de ventas, clientes, productos y generación de facturas electrónicas.

---

## 🛠️ Tecnologías utilizadas

### 📂 Base de datos
- Relacional **PostgreSQL**

### 🔗 API
- **Spring Boot**
- **REST**

### 🎨 Frontend
- **React**

### 🔒 Autenticación
- **OAuth 2.0 (Simulado con JWT)**
- **OpenID Connect (Simulado con JWT)**

### 📁 Repositorio
- **GIT**

---

## ✨ Características principales
- 📑 Gestión de facturas (creación, edición, cancelación).
- 👥 Administración de clientes y usuarios.
- 📦 Control de productos y servicios.
- 💳 Métodos de pago (efectivo, tarjeta, transferencia).
- 📈 Reportes financieros y de ventas.
- 🔐 Autenticación segura con **JWT**.

---

## 🚀 Puesta en Marcha del Backend

### 1. Requisitos de Entorno (JDK 21 y Maven)

Es **fundamental** que los siguientes componentes estén instalados y correctamente configurados en las variables de entorno:

| Componente | Versión Mínima | Configuración Clave |
| :--- | :--- | :--- |
| **Java Development Kit** | **JDK 21** | Configurar `JAVA_HOME` y añadir `%JAVA_HOME%\bin` al `Path` del sistema. |
| **Maven** | 3.8 o superior | Necesario para compilar el proyecto y gestionar dependencias. |
| **PostgreSQL** | N/A | El servidor de base de datos debe estar corriendo localmente. |
